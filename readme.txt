Web project template with basic User entity.

Main technologies:

    Spring Boot
    Spring Data
    Spring MVC
    Spring Security
    Query DSL
    Thymeleaf

For Intellij Idea

1) Go to Preferences -> Build, Execution, Deployment -> Annotation Processors
	Check Enable annotation processing checkbox
	In "Store generated sources relative to:" select Module content root
2) Add folder "src/main/generated" as sources
